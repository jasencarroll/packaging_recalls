# CLAUDE.md

## Project Overview

FDA packaging recalls data pipeline. Extracts drug recall data from the openFDA API, transforms and analyzes packaging-related defects, and serves an interactive Streamlit dashboard.

## Commands

```bash
uv sync                                        # Install dependencies
uv run python 1_data_pipeline/pipeline.py      # Run the data pipeline (fetch from FDA API)
uv run python 2_data_analysis/analysis.py      # Run analysis and generate visualizations
uv run streamlit run 3_data_dashboard/app.py   # Launch the Streamlit dashboard
uv run ruff check .                            # Lint
uv run ruff format .                           # Format
uv run mypy .                                  # Type check
```

## Key Files

- `1_data_pipeline/pipeline.py` -- FDA API extraction and transformation pipeline (`FDARecallPipeline`)
- `2_data_analysis/analysis.py` -- Data cleaning, risk scoring, visualization generation (`RecordTable`)
- `3_data_dashboard/app.py` -- Interactive Streamlit dashboard with Plotly charts
- `1_data_pipeline/fda_recall_data/` -- Raw and processed CSV/JSON output from the pipeline
- `2_data_analysis/database.db` -- SQLite database consumed by the dashboard
- `2_data_analysis/visualizations/` -- Static PNG charts generated by the analysis step
